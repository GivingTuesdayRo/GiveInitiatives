A plugin for Giving Tuesday websites to list give initiatives


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/give-initiatives` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
